## Variables and constans

### Inferred type
```swift
var languageName = "Swift" // variable languageName inferred as String
let languageName = "Swift" // constant
```
### Not inferred type
```swift
var languageName: String = "Swift" // variable
let languageName: String = "Swift" // constant
```
